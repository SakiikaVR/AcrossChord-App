# コードノート (Chord Note)

ブラウザだけで動く、モバイル向けのコード譜作成・閲覧 Web アプリです。
サーバー不要・アカウント不要で、データはすべて端末の `localStorage` に保存されます。

**▶ デモ (GitHub Pages): https://sakiikavr.github.io/chord-note/**

## 機能

### コード譜ライブラリ
- 曲の一覧表示（五十音・アルファベットのインデックス付き）
- 曲名・アーティスト名・本文でのインクリメンタル検索
- 長押しで複数選択モードに入り、まとめて削除
- プレイリストの作成・曲の追加/削除・まとめて削除

### コード譜エディタ（ChordPro 風記法）
- `{title: 曲名}` `{artist: アーティスト}` `{key: C}` でメタ情報を指定
- `{c: コメント}` でコメント行を表示
- 歌詞中に `[C]` `[Am7]` `[Eb/G]` のようにコードを埋め込み
- `[N.C.]` や `[----]` などの非コード記号にも対応

### コード譜ビューア
- 歌詞の上にコード名を配置して表示
- **コード名をタップするとギターの押さえ方（ダイアグラム）をポップアップ表示**
- ダイアグラムを常に表示するモードも選択可能
- 表示モード切替: コード名 / パワーコード (5th) / NNS（度数）表記
- Key の変更・Capo (0–11) 指定による自動移調
- シャープ (#) / フラット (♭) 表記の切替
- ズーム (50%–300%)
- 速度調整つきオートスクロール
- ブラウザの「戻る」ボタンに対応した画面遷移

### ギターチューニング対応
コードダイアグラムは選択中のチューニングに合わせて自動変換されます。
- Standard / Drop D / 半音下げ / Drop C# / 全音下げ / Drop C / Drop B

### クロマチックチューナー
- マイク入力から自己相関法でピッチを検出
- 音名・周波数・セント単位のズレをメーター表示

### データ管理
- ライブラリを JSON ファイルとしてエクスポート
- JSON ファイルからのインポート（マージ）
- 全データ削除

## 使い方

1. https://sakiikavr.github.io/chord-note/ を開く（スマホの場合は「ホーム画面に追加」推奨）
2. 右下の **＋** ボタンで新規曲を作成
3. エディタに以下のように入力して保存:

```
{title: ふるさと}
{artist: 文部省唱歌}
{key: C}

[C]うさぎ[F]おいし[G7]かのや[C]ま
```

初回起動時には、パブリックドメインのサンプル曲「ふるさと」（作詞: 高野辰之 / 作曲: 岡野貞一）が登録されています。

## ローカルでの実行

ビルド不要です。リポジトリを clone して `index.html` をブラウザで開くだけで動作します。

```
git clone https://github.com/SakiikaVR/chord-note.git
```
